Welcome to my Flask Library Management System! This web application is designed to help libraries manage their book inventory efficiently and users to easily find and request books.

Features

  • User Registration and Authentication: Users can create accounts, log in, and manage their profile.
  • Search Functionality: Users can search for books or sections based on various criteria.
  • Section Management: Librarians can manage library sections, including adding, updating, and deleting sections.
  • Book Management: Librarians can add, update, and delete books, along with uploading PDF files for each book.
  • Book Requests: Users can request books and view their pending requests, issued books, and completed books.
  • Admin Dashboard: Provides statistics and visualizations for administrators to monitor system activity.
  • User Dashboard: Gives users an overview of their book-related activities.
  • Password Change: Users can change their passwords securely within the application.

API Endpoints

  • /api/section - Endpoint for managing library sections.
  • /api/book - Endpoint for managing books.
  • /api/graph/bar_chart - Endpoint for generating a bar chart.
  • /api/graph/pie_chart - Endpoint for generating a pie chart.

Technologies Used

  • Flask: A micro web framework for Python.
  • SQLAlchemy: An SQL toolkit and Object-Relational Mapping (ORM) library for Python.
  • Flask-Login: A Flask extension for managing user sessions.
  • Flask-WTF: A Flask extension for handling web forms.
  • bcrypt: A password hashing library.
  • Matplotlib: A plotting library for creating visualizations.
  • Seaborn: A statistical data visualization library based on Matplotlib.
